# Conor's Portfolio

Based on the [PortfoliYOU](https://github.com/yousinix/portfolYOU) template, set up following the docs [here](https://yousinix.github.io/portfolYOU/docs/)

## Local development
To set up the local environment, run:
```bash
# install rbenv using pkg manager; then run rbinit

# then:
rbenv install 3.3.0
rbenv global 3.3.0
ruby -v  # confirm it shows 3.3.0
gem install bundler
bundle install
```

Then, to serve the site locally, run `bundle exec jekyll serve`
